# NewsForge

## Live Demo 
https://news-forge-alpha.vercel.app/

A full-stack MERN application that scrapes top stories from Hacker News and allows users to bookmark stories using JWT authentication.

## Features

- Hacker News scraper
- JWT Authentication
- Bookmark stories
- Protected routes
- React Context API
- MongoDB database
- Dark theme UI

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT

## Setup

### Backend

cd Backend
npm install
npm run dev

### Frontend

cd Frontend
npm install
npm run dev
